# Awesome-list Submission Descriptions

## awesome-llm

`kronaxis-router` - Open-source Go LLM proxy that auto-classifies prompts and routes to the cheapest capable model. Quality validation loop, per-service budgets, LoRA routing, response caching. Single binary, Apache 2.0.

## awesome-self-hosted

`Kronaxis Router` - Cost-optimised LLM routing proxy. Auto-classifies requests and routes to appropriate model tier (local or cloud). Failover, caching, budget enforcement, Prometheus metrics, embedded web UI. Single Go binary with no dependencies. `Apache-2.0` `Go`

## awesome-go

`kronaxis-router` - LLM inference proxy with automatic prompt classification, tiered model routing, quality validation, response caching, and per-service cost budgets. Supports LoRA adapter routing and 7 cloud providers as fallback. 70 tests. Apache 2.0.
